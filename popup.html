TikTok Download Buttons
Decides if newly opened tabs (through the fallback) should be auto-focused on or not.
Subfolder
Subfolder to download videos to. Leaving it empty will download videos to your default directory.
Naming
Naming template to use when saving videos. This will only have an effect when using the API. See templating.
Tips
Uncheck "Ask where to save each file before downloading" in chrome://settings/downloads for better download support.
